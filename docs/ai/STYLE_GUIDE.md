# STYLE GUIDE

## Code
- Python 3.11+; format with Black; type-check with mypy (where possible).
- Conventional commits; small PRs with clear scope.
- Avoid leaking secrets; load via Secret Manager.

## Docs
- Use concise headers, short paragraphs, tables when helpful.
- Keep PROJECT_BRIEF.md to ~1 page.

## Branding
- Primary: #008080
- Action:  #F97316
- Text: Warm Gray on Off-White
