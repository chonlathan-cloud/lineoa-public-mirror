# GLOSSARY

- **A (Company)**: Our organization operating the platform.
- **B (Business owner)**: Client who owns an OA we set up.
- **C (Customer)**: End-customer inside B’s OA.
- **MIA**: Admin OA (A→B).
- **Consumer OA**: OA that B uses to talk to C (B→C).
- **LIFF**: LINE Front-end Framework (with LINE Login/OpenID).

**Thai terms used in code/UI**
- “เริ่มต้นใช้งาน” / “เริ่มต้นเปิดร้านของคุณ”: Start/onboarding commands.
- “ยืนยันสิทธิ์เจ้าของร้าน”: Owner-binding prompt.
